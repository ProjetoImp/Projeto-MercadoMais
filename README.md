# Projeto-MercadoMais
O Mercado+ é um projeto de finalização de curso desenvolvido na Faculdade Impacta como requisito para conclusão da graduação.  A aplicação foi criada com o objetivo de oferecer uma solução prática para a gestão de mercados e pequenos comércios, permitindo o controle eficiente de produtos, estoque e vendas.

## Guia de Configuração e Execução da Aplicação

### Passo 1: Baixar e Instalar o Visual Studio 

1. Acesse o [site de download do Visual Studio](https://visualstudio.microsoft.com/pt-br/downloads/).
2. Selecione a versão "Community" e faça o download.
3. Siga as instruções de instalação.
4. 

### Passo 2: Clonar o Repositório

1. Abra o Visual Studio.
2. Clique em **Clonar Repositório**.
3. Insira o link do repositório e siga as instruções para clonar o projeto para o seu ambiente local.

### Passo 3: Baixar e Instalar o IntelliJ

1. Acesse o [site de download do IntelliJ]([https://nodejs.org/en/download/prebuilt-installer/current](https://www.jetbrains.com/idea/download/?section=windows)).
2. Baixe a versão Community Edition.
3. Execute o instalador e siga as instruções para completar a instalação.

### Passo 4: Configuração do backend

Após abrir o projeto no IntelliJ IDEA:

1. Verifique se o arquivo SupermercadoApplication está localizado no caminho:
 ```bash
src > main > java > com.example.supermercado
```
3. Caso o diretório não esteja configurado corretamente como código Java:
 ```bash
Clique com o botão direito em src/main/java
```
4. Selecione Mark Directory as → Sources Root
5. Verifique se o arquivo pom.xml está configurado como Maven Project.
6. Se não estiver, clique com o botão direito no arquivo pom.xml
7. Selecione Add as Maven Project
8. Após a configuração, compile e execute a aplicação diretamente pelo IntelliJ.
